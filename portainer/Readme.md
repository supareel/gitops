## System

- Ubuntu 22.0
- Ram 4GB
- CPU 4Core

## Steps

1. Configure DNS with `service_sub_domain.supareel.com` EC2 public IPs
2. Configure `nginx.conf` for domain declaration
3. Start docker-compose file `docker compose up`
4. visit `service_sub_domain.supareel.com`
